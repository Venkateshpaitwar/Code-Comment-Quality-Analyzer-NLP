# Code Comment Quality Analyzer (NLP)

This project analyzes the quality of code comments in Python files using NLP techniques.

## Features

- Extract comments using Python AST
- Clean and preprocess text
- Classify comment types
- Score comment quality
- Generate analysis reports

## Technologies

- Python
- NLP (NLTK / spaCy)
- Scikit-learn
- Pandas

## Project Workflow

1. Extract comments from Python files
2. Clean comments using NLP
3. Classify comment types
4. Score comment quality
5. Generate reports
